# Interactive Navigation Page

## Project Overview
This project is a modern and responsive multi-section website designed using HTML, CSS, and JavaScript.  
It includes interactive navigation, animated effects, blurred technology-themed background images, and responsive layouts.

The website contains four main sections:

- Home
- About
- Services
- Contact

Each section includes interactive buttons, smooth scrolling, glassmorphism effects, and animated backgrounds.

---

# Features

## Responsive Navigation Bar
- Fixed navigation bar
- Smooth hover effects
- Background color changes while scrolling
- Fully responsive for mobile devices

## Interactive Sections
- Full-screen sections
- Smooth scrolling navigation
- Hover animations
- Floating animated shapes

## Technology Background Images
- Computer and coding related backgrounds
- Blurred cinematic effect
- Zoom animation on background images

## Modern UI Design
- Glassmorphism cards
- Gradient buttons
- Calm and modern color palette
- Professional layout

## Responsive Design
- Works on desktop
- Tablet friendly
- Mobile responsive
